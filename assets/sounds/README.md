# Sound Files

This directory contains sound files for the Hair Clipper Prank app.

## Required Sound Files:

### Hair Clipper Sounds:
- clipper1.mp3 - Classic clipper sound
- clipper2.mp3 - Professional trimmer sound  
- clipper3.mp3 - Electric razor sound
- clipper4.mp3 - Cordless clipper sound

### Hair Dryer Sounds:
- dryer1.mp3 - Professional dryer sound
- dryer2.mp3 - Compact dryer sound
- dryer3.mp3 - Ionic dryer sound

### Funny Sounds:
- dog_bark.mp3 - Dog barking sound
- cat_meow.mp3 - Cat meowing sound
- fart.mp3 - Fart sound effect
- clown_horn.mp3 - Clown horn sound
- cow_moo.mp3 - Cow mooing sound
- duck_quack.mp3 - Duck quacking sound

## Note:
In a production app, you would place actual .mp3 or .wav sound files in this directory.
For development purposes, the app will attempt to load these files but will handle missing files gracefully.
